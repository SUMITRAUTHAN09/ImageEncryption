🔐 Image Encryption & Decryption



How to use

Choose an Image

Click on “Choose File”. Select the image you want to encrypt or decrypt (formats like .png, .jpg, .jpeg are supported).

Enter a Key

In the “Enter key” box, type any numeric key (e.g., 1234). ⚠️ Remember this key, because you’ll need the same key to decrypt the image later.

Encrypt / Decrypt

Click the “Encrypt / Decrypt” button. The system will process your image using the entered key. Download the Result A file named result.png will be downloaded automatically. If you encrypted the image → the result will look scrambled. If you decrypt it with the same key → you’ll get back the original image.
